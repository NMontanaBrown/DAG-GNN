# DAG-GNN

Adapting DAG-GNN to the synthetic data experiments in upcoming paper.

## Installing

```bash
conda create -n daggnn python=3.7
conda activate daggnn
conda install pytorch=1.10
pip install matplotlib pandas numpy seaborn
```
